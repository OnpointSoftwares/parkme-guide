Your comprehensive guide to using the ParkMe parking management system

Regular User Guide

Registration & Login
  1. Open the ParkMe app
  2. Click "Sign Up"
  3. Enter your email, password, and phone number
  4. Select role: User
  5. Click "Register"
  6. Login with your credentials
Tip: You can also sign in with Google for faster access!

Add Your Vehicle
  1. Navigate to "My Vehicles" tab
  2. Click "Add New Vehicle"
  3. Enter vehicle details:
    • Vehicle Number (e.g., KAA 123A)
    • Vehicle Title/Model
    • Owner Name
  4. Click "Save"
Vehicle number is required for bookings and violation tracking

Make a Reservation
  1. Go to "Dashboard"
  2. Browse available parking spots
  3. Select a parking location
  4. Choose:
    • Date
    • Check-in time
    • Check-out time
    • Vehicle
  5. Review total cost
  6. Click "Confirm Booking"
  7. Complete payment via M-Pesa
You'll receive a confirmation with booking details and QR code

View Your Bookings
  1. Navigate to "My Bookings" from profile menu
  2. Filter by: All, Upcoming, Completed, Cancelled
  3. Tap any booking to view details
  4. View QR code for check-in
Booking Status
  • Upcoming - Scheduled for future
  • Active - Currently parked
  • Completed - Finished
  • Cancelled - Canceled by owner
New Design: Modern dark theme with yellow accents for better visibility

Parking Owner Guide

Create a New Parking Spot
  1. Go to "My Spots" tab
  2. Click "Add Spot" button
  3. Fill in details:
    • Name (e.g., "Downtown Parking")
    • Address
    • City, State, Zip Code
    • Type (Open, Covered, Underground)
    • Size (Small, Medium, Large)
    • Total Spots
    • Cost per Hour (KES)
    • Description
    • Amenities (Security, CCTV, etc.)
  4. Set location on map
  5. Click "Save"

Edit or Delete Spots
  1. Tap on any parking spot card
  2. Use the menu (⋮) to:
    • Edit - Update details
    • Activate/Deactivate - Control visibility
    • Delete - Remove permanently
Deactivate spots temporarily instead of deleting to preserve booking history

View & Manage Reservations
  1. Go to "Bookings" tab
  2. View all bookings for your spots
  3. Filter by: All, Today, Upcoming, Completed
  4. Tap booking to view details
Cancel a Booking
  1. Open booking details
  2. Click "Cancel Booking"
  3. Enter cancellation reason
  4. Confirm cancellation
  5. Refund is automatically flagged
Always provide a clear reason when canceling bookings

View Performance Metrics
  1. Toggle Statistics icon in "My Spots"
  2. View:
    • Total Spots
    • Total Capacity
    • Currently Occupied
    • Available Spaces
    • Occupancy Rate
    • Active Spots
Enhanced UI: Dark theme with yellow accents, modern cards, and improved navigation

Kanjo Officer Guide

Complete Your Officer Profile
  1. First login will prompt profile setup
  2. Enter:
    • Badge Number (e.g., NCC-001)
    • Department (pre-filled: Parking Enforcement)
    • Zone Assignment (CBD, Westlands, etc.)
    • Contact Number (+254...)
  3. Click "Complete Setup"

Record a Parking Violation
  1. Click "Record Violation" on dashboard
  2. Select vehicle:
    • Choose from registered vehicles dropdown, OR
    • Enter vehicle number manually
  3. Set location:
    • Use "Get Current Location", OR
    • Enter manually
  4. Select violation type:
    • Illegal Parking
    • Expired Meter
    • No Parking Zone
    • Blocking Traffic
    • Double Parking
  5. Penalty amount auto-fills
  6. Add description (optional)
  7. Click "Submit Violation"
Violation is saved with timestamp and GPS coordinates

Access Violations List
  1. Click "All Violations" on dashboard
  2. View complete list with:
    • Vehicle number
    • Violation type
    • Penalty amount
    • Location
    • Timestamp
    • Payment status (Paid/Pending)

Status Indicators:

  • ✓ Paid - Violation settled
  • ⚠ Pending - Payment due

Add County-Managed Parking
  1. Click "Manage Spaces"
  2. Click "Add Space"
  3. Edit the created space:
    • Title
    • Location (click "Map" button)
    • Description
    • Capacity & Available spots
    • Hourly Rate (KES)
  4. Click "Save Changes"
Using Map Picker
  1. Click "Map" button next to location
  2. Tap anywhere on map to select location
  3. Drag marker for precise positioning
  4. Use "My Location" for current GPS
  5. Address auto-fills from coordinates
  6. Click "Confirm Location"
View Bookings
  1. Select a parking space
  2. Click menu → "View Bookings"
  3. Cancel bookings with reason if needed
Modern Interface: Dark theme with yellow accents, enhanced forms with rounded borders, and improved map picker

Generate Reports
  1. Click "Daily Report" on dashboard
  2. View summary of:
    • Total violations recorded
    • Total penalties issued
    • Payments collected
    • Outstanding fines

USSD Field Officer Guide

Overview

The USSD system allows field parking officers to verify customer payments using basic mobile phones - no smartphone or internet connection required!

Key Benefits:
  • Works on any mobile phone (feature phones included)
  • No internet connection needed
  • Instant payment verification
  • Record violations on the spot
  • Track daily performance
USSD Code

*384*55555#

Dial this code from your registered officer phone number

Verify if a Vehicle Has Paid
  1. Dial *384*55555#
  2. Select 1. Check Payment Status
  3. Enter vehicle registration number (e.g., KAA 123A)
  4. View instant results
Payment Status Results
PAID

Shows:

  • Vehicle number
  • Parking location
  • Date & time slot
  • Amount paid (KES)
  • Transaction ID

Action: Allow parking

NOT PAID / NO BOOKING

Shows:

  • Vehicle number
  • No booking found message

Action: Request payment or issue violation

Important: Always verify the vehicle number matches the physical license plate

See All Active Bookings
  1. Dial *384*55555#
  2. Select 2. View Today's Bookings
  3. View list of active bookings with:
    • ✓ Paid bookings
    • ✗ Unpaid bookings
    • Vehicle numbers
    • Locations

Shows up to 5 most recent bookings

Record Illegal Parking
  1. Dial *384*55555#
  2. Select 3. Report Issue
  3. Select 1. Illegal Parking
  4. Enter vehicle registration number
  5. Receive confirmation with:
    • Violation ID
    • Vehicle number
    • Penalty amount (KES 2,000)
    • Officer phone number
Other Issue Types
  • Payment Dispute - Customer claims payment not reflecting
  • System Error - Technical problems with the system
  • Other - General issues
All violations sync automatically with the Kanjo dashboard in the app

View Your Daily Performance
  1. Dial *384*55555#
  2. Select 4. My Stats
  3. View today's metrics:
    • Checks - Number of payment verifications
    • Violations - Violations recorded
    • Reports - Issues reported
    • Active Bookings - Total bookings today

Common Issues & Solutions
Issue: "System error. Please try again later."

Solutions:

  • Check your phone has network signal
  • Wait a few seconds and try again
  • Ensure you're using a registered officer phone number
Issue: No bookings found for valid vehicle

Solutions:

  • Verify vehicle number is entered correctly (no spaces)
  • Check if booking is for today's date
  • Customer may not have completed payment
Issue: USSD session timeout

Solutions:

  • Complete actions quickly (within 3 minutes)
  • Dial the code again to start fresh
  • Have vehicle numbers ready before dialing
