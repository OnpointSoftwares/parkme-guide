ParkMe User Documentation

Your comprehensive guide to using the ParkMe parking management system

Overview

ParkMe is a comprehensive parking management system with four user roles:

Regular User

Find and book parking

Parking Owner

Manage spots & bookings

Kanjo Officer

Enforce parking rules

Admin

System management

Regular User Guide

Registration & Login
  1. Open the ParkMe app
  2. Click "Sign Up"
  3. Enter your email, password, and phone number
  4. Select role: User
  5. Click "Register"
  6. Login with your credentials
Tip: You can also sign in with Google for faster access!

Add Your Vehicle
  1. Navigate to "My Vehicles" tab
  2. Click "Add New Vehicle"
  3. Enter vehicle details:
    • Vehicle Number (e.g., KAA 123A)
    • Vehicle Title/Model
    • Owner Name
  4. Click "Save"
Vehicle number is required for bookings and violation tracking

Make a Reservation
  1. Go to "Dashboard"
  2. Browse available parking spots
  3. Select a parking location
  4. Choose:
    • Date
    • Check-in time
    • Check-out time
    • Vehicle
  5. Review total cost
  6. Click "Confirm Booking"
  7. Complete payment via M-Pesa
You'll receive a confirmation with booking details and QR code

View Your Bookings
  1. Navigate to "My Bookings"
  2. Filter by: All, Today, Upcoming, Completed
  3. Tap any booking to view details
  4. View QR code for check-in
Booking Status
  • Pending - Awaiting confirmation
  • Confirmed - Ready for check-in
  • Checked In - Currently parked
  • Completed - Finished
  • Canceled - Canceled by owner

Parking Owner Guide

Create a New Parking Spot
  1. Go to "My Spots" tab
  2. Click "Add Spot" button
  3. Fill in details:
    • Name (e.g., "Downtown Parking")
    • Address
    • City, State, Zip Code
    • Type (Open, Covered, Underground)
    • Size (Small, Medium, Large)
    • Total Spots
    • Cost per Hour (KES)
    • Description
    • Amenities (Security, CCTV, etc.)
  4. Set location on map
  5. Click "Save"

Edit or Delete Spots
  1. Tap on any parking spot card
  2. Use the menu (⋮) to:
    • Edit - Update details
    • Activate/Deactivate - Control visibility
    • Delete - Remove permanently
Deactivate spots temporarily instead of deleting to preserve booking history

View & Manage Reservations
  1. Go to "Bookings" tab
  2. View all bookings for your spots
  3. Filter by: All, Today, Upcoming, Completed
  4. Tap booking to view details
Cancel a Booking
  1. Open booking details
  2. Click "Cancel Booking"
  3. Enter cancellation reason
  4. Confirm cancellation
  5. Refund is automatically flagged
Always provide a clear reason when canceling bookings

View Performance Metrics
  1. Toggle Statistics icon in "My Spots"
  2. View:
    • Total Spots
    • Total Capacity
    • Currently Occupied
    • Available Spaces
    • Occupancy Rate
    • Active Spots

Kanjo Officer Guide

Complete Your Officer Profile
  1. First login will prompt profile setup
  2. Enter:
    • Badge Number (e.g., NCC-001)
    • Department (pre-filled: Parking Enforcement)
    • Zone Assignment (CBD, Westlands, etc.)
    • Contact Number (+254...)
  3. Click "Complete Setup"

Record a Parking Violation
  1. Click "Record Violation" on dashboard
  2. Select vehicle:
    • Choose from registered vehicles dropdown, OR
    • Enter vehicle number manually
  3. Set location:
    • Use "Get Current Location", OR
    • Enter manually
  4. Select violation type:
    • Illegal Parking
    • Expired Meter
    • No Parking Zone
    • Blocking Traffic
    • Double Parking
  5. Penalty amount auto-fills
  6. Add description (optional)
  7. Click "Submit Violation"
Violation is saved with timestamp and GPS coordinates

Access Violations List
  1. Click "All Violations" on dashboard
  2. View complete list with:
    • Vehicle number
    • Violation type
    • Penalty amount
    • Location
    • Timestamp
    • Payment status (Paid/Pending)

Status Indicators:

  • ✓ Paid - Violation settled
  • ⚠ Pending - Payment due

Add County-Managed Parking
  1. Click "Manage Spaces"
  2. Click "Add Space"
  3. Edit the created space:
    • Title
    • Location (click "Map" button)
    • Description
    • Capacity & Available spots
    • Hourly Rate (KES)
  4. Click "Save Changes"
Using Map Picker
  1. Click "Map" button next to location
  2. Tap anywhere on map to select location
  3. Drag marker for precise positioning
  4. Use "My Location" for current GPS
  5. Address auto-fills from coordinates
  6. Click "Confirm Location"
View Bookings
  1. Select a parking space
  2. Click menu → "View Bookings"
  3. Cancel bookings with reason if needed

Generate Reports
  1. Click "Daily Report" on dashboard
  2. View summary of:
    • Total violations recorded
    • Total penalties issued
    • Payments collected
    • Outstanding fines

Admin Guide

System Administration
  1. Access admin dashboard after login
  2. Manage:
    • User accounts
    • Parking spot approvals
    • Owner registrations
    • System settings
Approve Parking Spots
  1. Navigate to "Approve Parking Spots"
  2. Review pending submissions
  3. Approve or reject with comments
Add Owner
  1. Click "Add Owner"
  2. Fill owner details form
  3. Submit for registration
